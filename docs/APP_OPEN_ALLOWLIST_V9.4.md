# ABU Native Helper V9.4 — App Open Allowlist Foundation

## Added

- Safe app open allowlist
- Telegram button
- WhatsApp button
- YouTube button
- Google Maps button
- Chrome button
- Gmail button
- Android Settings button
- Last app open request local log

## Important

This is local-only.
No server command pulls app actions.
No hidden launch.
No remote execution token.

## Manifest

Added package visibility queries for allowlisted apps only.

## Safety

Blocked:
- unknown package open
- silent hidden app open
- server-controlled app open
- dangerous execution

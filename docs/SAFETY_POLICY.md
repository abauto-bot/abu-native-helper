# ABU Native Helper Safety Policy

Allowed now:
- Local app screen
- Local ABU Phone ID generation
- Local pairing code save/clear
- Local phone status
- Battery read
- Storage read
- Device info read
- Local allowlisted app open button
- Notification listener skeleton disabled
- Notification Access settings shortcut
- Voice text draft local save
- Manual Telegram share intent
- Screen Vision policy section
- Display settings shortcut
- Accessibility service skeleton disabled
- Accessibility settings shortcut
- In-app ABU Launcher Shell cards
- Safe URL open buttons

Blocked now:
- Server command execution
- Unknown package launch
- Root
- Factory reset
- Delete personal data
- Hidden screenshot
- Silent screen recording
- Silent spying
- Bypass lock screen
- Notification reading
- Auto reply
- Microphone recording
- Always listening
- Auto-send Telegram message
- Accessibility tap/scroll/type
- Accessibility screen read
- Screen capture
- OCR/UI detection
- Android default launcher replacement
- CATEGORY_HOME launcher intent
- GOVO destructive actions
- Kasm modification

Future execution rule:
Observe → Risk classify → Audit → Confirm → Execute allowed action → Postcheck

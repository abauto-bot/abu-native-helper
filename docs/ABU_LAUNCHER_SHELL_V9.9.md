# ABU Native Helper V9.9 — ABU Launcher Shell Foundation

## Added

- ABU Launcher Shell section
- In-app quick command cards
- Open ABU Web Portal
- Open Pairing Portal
- Open GOVO Express
- Open Telegram
- Local launcher state display

## Important

V9.9 does not replace Android launcher.

Not added:
- CATEGORY_HOME
- default launcher intent
- background launcher service
- device admin
- lock screen control
- hidden automation

## Security

This is local-only UI.
No server command execution.
No dangerous permissions.
No hidden control.

# ABU Native Helper V9.3 — Phone Status Model

## Added

- Local ABU Phone ID display
- Pairing state display
- Device model/manufacturer/Android SDK
- Battery level and charging status
- Internal storage total/free/used
- Refresh Phone Status button

## Permissions

No new dangerous permissions.

Current manifest permission:
- INTERNET only

Battery status:
- Uses Android battery changed sticky broadcast.

Storage status:
- Uses local app/device data partition stats.

## Still Disabled

- Server sync
- Execution token
- Notification listener
- Screenshot
- Accessibility
- Dangerous phone actions
